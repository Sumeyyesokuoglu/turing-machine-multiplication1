# Turing Machine - Binary Multiplication Model

Bu proje, bilgisayarların **Aritmetik Mantık Birimlerinde (ALU)** gerçekleşen ikili çarpma (binary multiplication) işleminin matematiksel arka planını bir **Turing Makinesi** simülasyonu kullanarak somutlaştırmaktadır. 

Kod yapısı, operant ayrıştırma kurallarına uyarak çalışır. 
Her adımda Turing makinesinin kafa hareketlerini, anlık durumunu ve bant içeriğini çıktı olarak terminale yazdırır. Ayrıca kullanıcı `0` ve `1` dışında geçersiz bir girdi girdiğinde hata mesajı fırlatan bir kontrol mekanizmasına sahiptir.



## 🚀 Nasıl Çalıştırılır?

Projenin çalışması için bilgisayarınızda **Python 3.x** kurulu olması yeterlidir.

1. Bu repoyu bilgisayarınıza indirin veya kod dosyasını kopyalayın.
2. Terminali / Komut Satırını açıp dosyanın olduğu dizine gidin.
3. Aşağıdaki komut ile simülasyonu başlatın:
   ```bash
   python turing_carpma.py
